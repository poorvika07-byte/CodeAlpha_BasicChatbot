**# CodeAlpha_BasicChatbot**# Basic Rule-Based Chatbot

A simple chatbot developed using Python that responds to predefined user inputs with fixed responses.

## Features

- Responds to greetings
- Answers basic questions
- Exit command support
- User-friendly console interface

## Technologies Used

- Python
- Functions
- Loops
- Conditional Statements

## Supported Commands

hello
how are you
what is your name
bye

## How to Run

1. Download the project.
2. Open terminal or command prompt.
3. Run:

python chatbot.py

## Sample Conversation

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: I am CodeAlpha Bot.

You: bye
Bot: Goodbye!

## Concepts Implemented

- Functions
- if-elif-else
- While Loop
- User Interaction
